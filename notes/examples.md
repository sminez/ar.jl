# Examples Uses of the AR.jl module

### Creating and manipulating alphas
The building blocks of the algebra are the 16 α values that represent the point,
lines, areas, volumes and hyper-volume of G4.

```julia
julia> α("1") * α("2")
α12


julia> α("2") * α("1")
-α12
```
